# SL (Stealth Game Project)

본 프로젝트는 Unity 엔진을 활용하여 게임의 전체 루프를 설계하고 C# 스크립팅 능력을 배양하기 위해 개발된 소규모 인디 잠입 액션 게임입니다.

## 1. 개요
- **프로젝트 명**: SL
- **개발 기간**: 2023.01
- **사용 엔진**: Unity
- **개발 목적**: 잠입 매커니즘 구현, 적 AI 로직 설계, 전체 게임 사이클(시작-진행-기록) 경험

## 2. 핵심 게임 시스템

### 2.1 구역별 이원화 매커니즘
맵은 **밝은 구역**과 **어두운 구역**으로 나뉘며, 구역의 성격에 따라 탐지 및 시야 로직이 다르게 적용됩니다.

| 구분 | 적의 탐지 방식 | 플레이어 시야 정보 |
|:---|:---|:---|
| **밝은 구역** | 부채꼴 형태의 시야(Visual) 기반 탐지 | 적의 위치와 형태를 상시 확인 가능 |
| **어두운 구역** | 원형 형태의 청각(Acoustic) 기반 탐지 | 적의 이동 흔적을 통해 위치 추정 |

### 2.2 탐지 로직 및 AI 행동
- **기본 행동**: 지정된 위치에서 대기하거나 설정된 경로를 순찰합니다.
- **추적 알고리즘**: 플레이어 감지 시, 마지막으로 감지된 위치(Last Known Position)로 이동하여 대기합니다.
- **구역별 탐지 특성**:
    - **밝은 구역**: 탐지 범위가 넓으나 발견 후 추적까지 일정 시간의 지연이 발생합니다.
    - **어두운 구역**: 탐지 범위는 좁으나 감지 즉시 플레이어 위치를 추적합니다.
- **상태 표시**: 화면 상단의 `Detected` 텍스트 UI를 통해 감지 여부를 확인합니다.

## 3. 게임 구성 요소

### 3.1 아이템 시스템
상호작용은 아이템 소지 시 해당 오브젝트에 접촉하면 자동으로 수행됩니다.

- **열쇠 (Key)**: 잠긴 문(자물쇠 표시) 통과 시 소비됩니다.
- **별 (Star)**: 수집 요소입니다. 맵 안의 모든 별을 획득한 상태로 클리어해야만 소요 시간 기록이 유효하게 처리됩니다.
- **칼 (Knife)**: 적과 접촉 시 소모하여 적을 처치합니다. 미소지 상태에서 적과 접촉할 경우 게임 오버 처리됩니다.

### 3.2 인터페이스 (UI)
- **정보 노출 최소화**: 게임 몰입을 위해 상시 UI는 생략되어 있습니다.
- **상태 확인**: `Q` 키를 입력하고 있는 동안 하단 패널이 활성화되어 현재 시간 및 보유 아이템 목록을 표시합니다.
- **터치 지원**: 모바일 환경을 고려한 터치 인터페이스가 구현되어 있습니다. (테스트 단계)

## 4. 조작법
- **이동**: 키보드 방향키 또는 화면 터치 조작
- **정보 확인**: `Q` (Hold)
- **상호작용**: 오브젝트 접촉 시 자동 수행
